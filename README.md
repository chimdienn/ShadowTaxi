## ğŸ–¥ï¸ How to Run

1. Open the project in **IntelliJ IDEA** (or another Java IDE).  
2. Navigate to: src/ShadowTaxi.java
3. Run the `main` method.  
4. The game window will appear:
- Enter your **username** on the player info screen.
